## CDPSfy

### Dependencies

- node.js
- npm

### How to install

<pre>
git clone https://github.com/oscaryhusse/CDPSfy.git
cd CDPSfy
npm install
</pre>

### How to configure

+ The server is configured to listen in port 80.

### How to run

<pre>
npm start
</pre>

#### Default music: http://www.bensound.com
